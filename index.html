What is Frontend

The frontend is the visible part of a website or web application that users directly see and interact with.

This includes everything from buttons, images, forms, and the layout of the page to colors, fonts, and animations. It’s what makes a website look attractive and usable.

<div class="container">Hello World</div>
  • User Interface (UI)
  • Technologies used: HTML, CSS, JavaScript
    • About HTML

    HTML (HyperText Markup Language) is the basic building block of web pages. It structures content like text, images, links, buttons, and more.

    Think of HTML as the skeleton of a webpage – it tells the browser what each part of the content is and how it’s organized.

    <h1>My Website</h1>
  • Used for Headings (h1 to h6)
  • Used for Paragraphs, Lists, Links
  • Gives the page structure
    • About CSS

    CSS (Cascading Style Sheets) is used to add style to HTML content. It controls the colors, fonts, spacing, borders, and layout of the page.

    While HTML gives the structure, CSS makes the webpage look nice and visually appealing to the user.

    body { background-color: lightblue; }
  • Adds styles like color, font, and spacing
  • Controls layout using Flexbox and Grid
    • About JavaScript

    JavaScript is a programming language that adds interactivity to websites. It helps the website respond to user actions like clicks, typing, or scrolling.

    With JavaScript, you can create animations, update content without refreshing, validate forms, and much more.

    alert("Hello World!");
  • Handles dynamic behavior
  • Useful for things like sliders, pop-ups, and forms
    • About Frontend frameworks

    Frontend frameworks help developers build websites faster and in a more organized way by providing pre-written code and tools.

    Popular frameworks like React, Angular, and Vue let you reuse components (like buttons, forms, etc.) across different parts of your website, which saves time and keeps your code clean.

    const element = <h1>Hello</h1>;
  • Encourage component-based architecture
  • Improve development speed and code organization
